# Comment ajouter l'image de fond

## Option 1 : Image locale (Recommandé)

1. **Télécharge ou copie** la photo de l'événement (la conférence avec le public)

2. **Renomme l'image** en : `conference-background.jpg`

3. **Place l'image** dans le dossier :
   ```
   public/images/conference-background.jpg
   ```

4. **Redémarre le serveur** (`npm run dev`) et l'image apparaîtra automatiquement

## Option 2 : URL externe

Si l'image est déjà hébergée en ligne, donne-moi l'URL et je l'ajouterai dans le code.

## Option 3 : Utiliser une image depuis un service

Si tu as l'image sur :
- Google Drive
- Dropbox
- Imgur
- Unsplash
- Ou tout autre service

Donne-moi le lien et je l'intégrerai.

## Format recommandé

- **Taille** : 1920x1080 pixels ou plus
- **Format** : JPG (recommandé) ou PNG
- **Poids** : Optimisé pour le web (max 500KB)

Une fois l'image ajoutée, elle apparaîtra automatiquement en arrière-plan de la section "Prêt à Transformer Votre Vision en Réalité?" avec un overlay sombre pour la lisibilité du texte.
