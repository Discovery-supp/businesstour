# Business Tours

Plateforme Business Tours (Vite + React + TypeScript) avec formulaire de réservation, intégration Supabase, et paiements Stripe (mode test).

## Démarrage

```bash
npm install
npm run dev
```

## Configuration

- Supabase : voir `SETUP_SUPABASE.md`
- Stripe : voir `SETUP_STRIPE.md`
